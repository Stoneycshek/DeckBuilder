# 元素合成 Deckbuilder

> **Roguelike × Deckbuilder** - 卡牌即技能,亦是可拆解重組的零件

## 專案概述

這是一款創新的 Roguelike 卡牌遊戲,核心特色是卡牌的雙重身份:
- **戰鬥中**:作為技能直接使用
- **戰鬥外**:可拆解成零件並合成新卡牌

## 核心理念

> 與其追求單卡平衡,不如確保「選擇本身有代價」

玩家需在以下選擇間權衡:
- 保留卡牌作為即戰力
- 拆解卡牌換取長期強化

## 技術棧

- **前端**: React 18 + TypeScript 5
- **構建工具**: Vite
- **樣式**: CSS Modules / Tailwind CSS
- **數據管理**: JSON 配置文件
- **存檔**: LocalStorage (MVP 階段)

## 核心玩法

### 遊戲循環

1. 進入戰鬥
2. 使用卡牌進行回合制戰鬥
3. 獲得卡牌/資源獎勵
4. 進入合成商店
5. 拆解/合成/強化卡牌
6. 進入下一場戰鬥

### 基礎系統

- **12 張基礎卡**: 火🔥 / 冰❄ / 物理⚔ 三元素
- **合成系統**: 元素 × 型態 / 元素 × 元素 / 型態 × 修飾
- **戰鬥系統**: 能量制回合戰鬥 (3 能量/回合)
- **數值基準**: 1 能量 ≈ 5 點戰鬥價值
